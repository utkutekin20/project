## Güncelleme Yayınlama Komutu

Aşağıdaki promptu VS Code yapay zekasına yapıştır:

```
GÜNCELLEME YAYINLA: package.json'daki version'ı bir artır (patch), npm run build yap, sonra GH_TOKEN environment variable'ını ayarla ve npx electron-builder --win --publish always çalıştır. Ardından git add -A, git commit ve git push origin main yap.
```
